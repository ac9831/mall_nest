## About

Nest로 구현한 쇼핑몰 - Nest 공부 및 정리를 위해 쇼핑몰 구축이라는 주제로 Nest 프로젝트 진행합니다.

## Install

```bash
$ npm i
```

## Usage

```bash
$ npm run start
```

#### dev

```bash
$ npm run start:dev
```
