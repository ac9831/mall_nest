## About

Nest로 구현한 쇼핑몰 - Nest 공부 및 정리를 위해 쇼핑몰 구축이라는 주제로 Nest 프로젝트 진행합니다.

## Install

```bash
$ npm i
```

## Usage

```bash
$ npm run start
```

#### dev

```bash
$ npm run start:dev
```

## 참고 사항

1. 설치 및 실행은 위 방법 참고바랍니다.
2. mysql를 활용했습니다.
3. 이 프로젝트는 NestJS + TypeScript + TypeOrm를 사용했습니다.
4. port는 3000번이 기본입니다.
5. swagger 주소는 /api-docs 입니다.
